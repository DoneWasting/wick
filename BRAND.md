# Wick — Brand

**Name:** Wick
**Tagline:** Candle Timer for Traders

## Color tokens

| Token              | Hex       | Role                                                                                 |
| ------------------ | --------- | ------------------------------------------------------------------------------------ |
| Background         | `#0E0E0E` | App background                                                                       |
| Bullish green      | `#1FBF7A` | "On" state, positive/active actions, next-event indicators                           |
| Bearish red        | `#E24B4A` | "Off" state, destructive actions                                                     |
| Text primary       | `#FFFFFF` | Headings, primary copy                                                               |
| Text secondary     | `#9A9A9A` | Captions, helper text, inactive icons                                                |
| Neutral accent     | `#4a90e2` | Selected radio/checkbox; non-destructive dismiss actions (Cancel). Avoid elsewhere. |
